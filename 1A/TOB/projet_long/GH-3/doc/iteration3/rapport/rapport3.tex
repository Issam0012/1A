\section{Élaboration importance des fonctionnalités}
Pour élaborer le classement de l'importance des fonctionnalités a implanter nous avons utiliser le modèle MoSCoW (Must / Should / Could / Won't).

\subsection{M de MoSCoW}

Liste des différents éléments classifiés comme Must:\\
\begin{itemize}
	\item Interface de création de personnage
	\item Interface de création d'environnement
	\item Interface de jeu
\end{itemize}

\subsection{S de MoSCoW}

Liste des différents éléments classifiés comme Should:\\
\begin{itemize}
	\item Implantation des dés
	\item Menu Principal
\end{itemize}

\subsection{C de MoSCoW}

Liste des différents éléments classifiés comme Could:\\
\begin{itemize}
	\item Onglet de prise de note
	\item Interface de création carte d'événement
	\item Gestionnaire de dé
\end{itemize}

\subsection{W de MoSCoW}

Liste des différents éléments classifiés comme Won't:\\
\begin{itemize}
	\item Moteur qui lance de dé
\end{itemize}

\section{Élaboration des EPICs}

Liste des EPICs : \\

\begin{itemize}
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir éditer mon personnage \textbf{afin de} personnaliser les caractéristiques de celui-ci.
	\item \textbf{En tant que} maître du jeu \textbf{je veux} pouvoir éditer les environnements  \textbf{afin de} personnaliser l'agencement des éléments de celui-ci.
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir faire une partie \textbf{afin d'}utiliser mes créations.
\end{itemize}

\subsection{User storys de l'EPIC 1}
Voici les différentes user story liées a la première EPIC de la liste : \\

\begin{itemize}
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir éditer mon personnage \textbf{afin de} modifier les statistiques (force, intelligence, ...)  de celui-ci.	
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir éditer mon personnage \textbf{afin d'}équiper mon personnage comme je le souhaite.
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir éditer mon personnage \textbf{afin de} choisir la race de celui-ci.\\
\end{itemize}

De plus afin de vérifier avec des tests unitaires le bon fonctionnement des fonctionnalités implantées nous devons respecter les détails suivants : \\

\begin{itemize}
	\item Vérifier que les modifications sont bien enregistrées
	\item Vérifier que le personnage créé soit bien présent sur l'interface
\end{itemize}

\subsection{User storys de l'EPIC 2}

Voici les différentes user story liées a la deuxième EPIC de la liste : \\

\begin{itemize}
	\item \textbf{En tant que} maître du jeu \textbf{je veux} pouvoir éditer les environnements  \textbf{afin de} rajouter des éléments de décor a celui-ci.
	\item \textbf{En tant que} maître du jeu \textbf{je veux} pouvoir éditer les environnements  \textbf{afin de} rajouter des ambiances sonores.\\
\end{itemize}

\subsection{User storys de l'EPIC 3}

Voici les différentes user story liées a la troisième EPIC de la liste : \\

\begin{itemize}
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir faire une partie \textbf{afin de} pouvoir déplacer mon personnage.
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir faire une partie \textbf{afin de} pouvoir intéragir avec les éléments du décor.
	\item \textbf{En tant que} joueur \textbf{je veux} pouvoir faire une partie \textbf{afin de} pouvoir interagir avec les autres joueurs.\\

\end{itemize}
